📊 User Flow Overview

The AGAA prototype follows a simple, intuitive 6-screen flow designed for ease of use.

1. Home
→
2. Describe
→
3. Eligibility
→
4. Schemes
→
5. Form Review
→
6. Confirmation

🎯 Design Principles

  • Mobile-First: Optimized for mobile devices (320px+)
  • Simple: Minimal cognitive load for all users
  • Visual: Icons and colors for better understanding
  • Progressive: Step-by-step guidance
  • Accessible: WCAG 2.1 AA compliant design

🎨 Color Palette

  • Primary: #a8e6cf (Pista Green - Growth, Development)
  • Secondary: #7ed6a8 (Dark Green - Success, Progress)
  • Accent: #5cb85c (Action Green)
  • Background: #F5F5F5 (Light Gray)
  • Text: #2d3748 (Dark Gray)

✨ Key Features

  • Multi-language support (English, Hindi, Tamil)
  • Dual use cases (Scholarship & Farmer Welfare)
  • AI-powered form generation (Scholarship flow)
  • Client-side processing (Farmer flow)
  • Responsive design for all devices
  • Progress indicators throughout the journey
