💡 Our Solution: AGAA

An AI-powered web application that automatically fills government forms using natural language input

🤖 AI-Powered

Uses Amazon Bedrock (Claude 3 Haiku) to understand user needs and auto-fill forms

🌐 Multi-Lingual

Supports English, Hindi, and Tamil for inclusive access

📱 Mobile-First

Responsive design works on any device, even with poor connectivity

✅ Validation

Real-time validation prevents errors and rejections

⚙️ How It Works

1
User Input: Citizen describes their need in simple language (text or voice)
2
AI Analysis: Amazon Bedrock analyzes intent and identifies the right scheme
3
Eligibility Check: System verifies if user qualifies for the scheme
4
Auto-Fill: AI automatically fills the application form with user data
5
Review & Submit: User reviews, uploads documents, and submits
6
Confirmation: Receive confirmation number and next steps
