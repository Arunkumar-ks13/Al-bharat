📋 Architecture Specifications

🎯 Key Features

  • Serverless Architecture: No server management, auto-scaling
  • Cost-Effective: Pay only for what you use (~$0.0018 per transaction)
  • Highly Available: 99.9% uptime with AWS managed services
  • Secure: IAM roles, HTTPS, encrypted storage
  • Scalable: Handles 1000+ concurrent users

AWS Services Used

Service Purpose Configuration
S3 Static Website Frontend Hosting agaa-frontend-1772860549
API Gateway REST API Endpoint phxb8mfmqk.execute-api.us-east-1.amazonaws.com/prod
Lambda Functions Backend Logic 3 functions, Node.js 20.x, 512 MB, 30s timeout
AWS Bedrock AI Processing Claude 3 Haiku (anthropic.claude-3-haiku-20240307-v1:0)
DynamoDB Database agaa-applications (On-Demand)
S3 Bucket Document Storage agaa-docs-ak-1772860173
CloudWatch Monitoring & Logs Lambda logs, API Gateway logs, metrics
IAM Security & Access Lambda execution roles, service permissions

Lambda Functions

Function Endpoint Purpose
process-user-input POST /process-input Process user eligibility, analyze requirements
generate-form POST /generate-form Generate application form using AI
submit-application POST /submit Submit application, generate PDF, store in S3
