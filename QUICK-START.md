## 📱 How to Test

### Test Scenario 1: Education Scholarship
1. Open the live URL
2. Select language (English/Hindi/Tamil)
3. Click "Get Started"
4. Enter: "I need a scholarship for engineering"
5. Answer 10 eligibility questions
6. View recommended schemes
7. Generate and download form

### Test Scenario 2: Farmer Welfare
1. Open the live URL
2. Select language (try Tamil)
3. Click "Get Started"
4. Enter: "I am a farmer looking for fertilizer subsidy" or "உழவர்" (Tamil keyword)
5. View 6 farmer welfare schemes
6. Select a scheme
7. Fill the application form
8. Download the form

### Test Scenario 3: Multi-Language
1. Select Hindi or Tamil
2. Complete the flow
3. Download PDF
4. Verify PDF is in selected language

## 📊 Technical Highlights

### Frontend
- Pure JavaScript (no frameworks)
- Responsive design
- Client-side validation
- Progressive enhancement

### Backend
- 3 Lambda functions
- DynamoDB for storage
- S3 for documents
- API Gateway for REST API
- Bedrock for AI

### Cost
- Prototype: $14-20/month
- Production: ~$755/month (1000 forms/day)
